# CoreVault
- můj hlavní repozitář zápisek, určeno pro [Obsidian](https://obsidian.md/)
- závisí na následujících Obsidian pluginech:
	- [Extended MathJax](https://github.com/wei2912/obsidian-latex)
		- umožňuje použití `preamble.sty` na vlastní LaTeXové zápisy
		- hojně to používám, takže tenhle fakt povinný
	- [Folder notes](https://github.com/LostPaul/obsidian-folder-notes)
		- dovoluje pěknější strukturu zápisek
		- jde to i bez něj, ale navigace otravnější
- jde trochu zobrazit i na GitHub, ale čím je to matematičtější, tím hůř se LaTeX zobrazuje, GitHub nepodporuje `preamble.sty` a renderuje LaTeX trochu jinak; tohle je fakt dělaný na stažení a zobrazení v Obsidianu
